# Learning Journey: What are fractions? (MVP)

## Module Overview
- **Target Audience**: Grade 4 students (Indonesian/Asian)
- **Total Duration**: ~20 minutes
- **Voice Agent Name**: "Math Mate"
- **Flow**: Fixed sequence, no branching
